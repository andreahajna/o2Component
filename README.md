# O2 Component

React application using TypeScript and Vite. 

## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

1. Clone the repository
2. Navigate to the project directory:
    ```sh
    cd o2-component
    ```
3. Install the dependencies:
    ```sh
    npm install
    ```

### Scripts

- **Development**:
    ```sh
    npm run dev
    ```
